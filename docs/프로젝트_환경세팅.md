# 📌 프로젝트 환경 세팅 가이드

## 1. Conda 환경 생성

프로젝트 루트에서 실행:

```
conda env create-f environment.yml
conda activate skn-1st
```

---

## 2. VSCode 설정

프로젝트 루트에 `.vscode/settings.json` 파일 생성

```
{
    "python-envs.defaultEnvManager":"ms-python.python:conda",
    "python-envs.defaultPackageManager":"ms-python.python:conda",

    "python.defaultInterpreterPath":"C:\\Users\\{윈도우 사용자 이름}\\miniconda3\\envs\\skn-1st\\python.exe", // ⚠️ 본인 환경에 맞게 수정 필수

    "python.terminal.activateEnvironment":true,

    // IDE import 인식 (src 기반 import 가능)
    "python.analysis.extraPaths": ["./src"],

    // .env 자동 로드
    "python.envFile":"${workspaceFolder}/.env",

    // 터미널 실행 시 PYTHONPATH 적용 (streamlit 포함)
    "terminal.integrated.env.windows": {
        "PYTHONPATH":"src"
    }
}
```

## 4. Streamlit 실행 방법

```
cd src
python-m streamlit run app/main.py
```

---

## ⚠️ 주의사항

- `python.defaultInterpreterPath`는 반드시 본인 PC 환경에 맞게 수정해야 합니다
- `.env` 파일은 Git에 포함하지 않습니다
- Streamlit은 반드시 `python -m` 방식으로 실행해야 import 오류가 발생하지 않습니다
